# 亚马逊每周热门选品报告（静态网页）

本项目是纯静态网页（HTML/CSS/JS），无需安装依赖或构建。

## 本地打开
- 直接用浏览器打开：`index.html`

## 发布到 GitHub Pages（推荐）
1. 登录 GitHub，新建一个 **Public** 仓库（例如：`amazon-weekly-report`）。
2. 将本目录下文件上传到仓库根目录：
   - `index.html`
   - `app.js`
   - `styles.css`
   - `.nojekyll`
3. 进入仓库 **Settings → Pages**：
   - Source 选择 **Deploy from a branch**
   - Branch 选择 **main**，Folder 选择 **/(root)**
   - 点击 **Save**
4. 等待 GitHub 部署完成后，Pages 会显示访问地址：
   - `https://<你的用户名>.github.io/<仓库名>/`

> 如果你的仓库名就是 `<你的用户名>.github.io`，那访问地址通常是：`https://<你的用户名>.github.io/`
